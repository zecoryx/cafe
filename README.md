# Cafe Management System

A web-based cafe management solution designed to streamline daily operations. Built with React and Vite, the platform provides dedicated interfaces for both administrators and staff to handle orders, track inventory, and monitor business performance efficiently.

## Core Features

- **Role-Based Access**: Secure authentication with distinct admin and cashier privileges.
- **Cashier Interface**: 
  - Point of Sale (POS) menu system
  - Active order management
  - Table status tracking
  - Transaction history
- **Administrator Dashboard**: 
  - Centralized business metrics and oversight
  - Employee management
  - Inventory tracking

## Tech Stack

- **Frontend**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **State Management**: React Context API

## Getting Started

### Prerequisites

Ensure you have Node.js installed on your local environment.

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
